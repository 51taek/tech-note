### 오브젝트
- 객체 지향의 개념 학습
- 객체 지향의 궁극적인 목표는 '변경하기 쉬운 코드를 만드는 것' 임을 학습
- 객체지향은 정답이 아님을 학습. 객체지향은 변경하기 쉬운 코드로 만들어주는 대신 복잡도를 향상된다. 때로는 변경 가능성에 비해 지나치게 코드 복잡도가 올라가는 경우도 분명히 있다
- 책임 단위의 설계 과정을 학습하고 왜 책임을 기준으로 객체를 나눠야 하는 것인지 책임의 범위를 어떻게 나눠야 하는 것인지를 학습
